# News Tracker

获取特定主题的最新新闻和动态信息。

## 主题类型识别

解析用户查询，识别主题类型：

| 类型 | 识别特征 | 示例 |
|------|----------|------|
| **领域** | 行业/学科名词 | AI、科技、金融、医疗、教育 |
| **公司** | 企业/组织名称 | OpenAI、Google、Microsoft、Tesla |
| **人物** | 人名 | 马斯克、Sam Altman、黄仁勋 |
| **地区** | 国家/地区名 | 中国、美国、欧洲、日本 |
| **产品** | 产品/服务名称 | ChatGPT、Claude、iPhone |
| **事件** | 活动/事件名 | CES 2026、WWDC、某发布会 |

## 意图识别

判断用户意图：

1. **新闻聚合**：用户想浏览某主题的最新新闻列表
   - 特征：使用"新闻"、"动态"、"资讯"等词
   - 示例："AI领域最新新闻"、"马斯克最近动态"

2. **智能问答**：用户想要一个基于最新信息的答案
   - 特征：疑问句，问"是什么"、"哪个"、"怎么样"
   - 示例："目前最强AI模型是哪个"、"OpenAI现在的估值是多少"

## 搜索执行

### 时间范围智能判断

根据主题特性自动选择搜索时间范围：

| 主题特性 | 时间范围 | 理由 |
|----------|----------|------|
| 热点人物/公司 | 24小时-1周 | 动态更新频繁 |
| 行业领域 | 1周-1个月 | 需要积累一定数量 |
| 特定事件 | 事件相关时间段 | 如展会期间 |
| 技术产品 | 1周-2周 | 版本发布相关 |

### 搜索策略

使用 `search_web` 工具执行搜索：

```
搜索关键词构造：
- 主题词 + "新闻"/"news"
- 主题词 + "最新"/"latest"
- 主题词 + 时间词（如"2026年1月"）
```

优先选择权威来源：
- 科技类：TechCrunch、The Verge、Wired、36氪、虎嗅
- 综合类：Reuters、BBC、新华网、财联社
- AI专业：arXiv、Hacker News、AI相关博客

### 搜索数量

默认获取约 **10 条** 相关新闻。

## 输出格式

### 新闻列表格式

每条新闻包含完整信息，使用中文输出（英文新闻翻译为中文）：

```markdown
### 1. [新闻标题]

- **摘要**：[2-3句话的新闻摘要]
- **来源**：[来源网站名称]
- **时间**：[发布日期，格式：YYYY-MM-DD]
- **相关性**：[⭐⭐⭐⭐⭐ 使用星级表示，1-5星]
- **链接**：[阅读原文](URL)
```

### 智能问答格式

对于问答类查询，先搜索最新信息，然后给出结构化回答：

```markdown
## 回答

[基于最新信息的直接回答]

## 信息来源

[列出支撑回答的新闻/信息来源]
```

## 异常处理

### 无相关结果

如果搜索不到相关新闻：

```markdown
未找到关于「[主题]」的近期新闻。

**建议**：
1. 尝试更换关键词，如「[替代关键词1]」「[替代关键词2]」
2. 扩大搜索范围，如从 24 小时扩展到一周
3. 确认主题名称拼写是否正确
```

### 结果较少

如果结果少于 5 条：

```markdown
以下是关于「[主题]」的最新新闻（共 N 条）：

[新闻列表]

> 💡 提示：该主题近期新闻较少，可尝试扩大时间范围或相关主题。
```

- **代码生成**：Claude 3.5 Opus 领先

- **成本效益**：GPT-4o-mini 性价比最高

## 信息来源

1. [AI模型2026年1月排行榜 - Hugging Face](https://...)
2. [Claude 3.5 Opus发布评测 - TechCrunch](https://...)
